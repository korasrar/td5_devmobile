## Lancer l'application

### Configuration Supabase

Modifier le fichier [./lib/services/DatabaseService.dart](./lib/services/DatabaseService.dart) avec vos identifiants :

```dart
try {
      await Supabase.initialize(
        url: 'url here',
        anonKey: 'anon key here',
      );
    } catch (e) {
      // ruuh
    }
```

### Commandes

- Installer les dépendances :

```bash
flutter pub get
```

- Builder les paquets SQLite pour le web :

```bash
dart run sqflite_common_ffi_web:setup --force
```

- Lancer sur Chrome :

```bash
flutter run -d Chrome
```
